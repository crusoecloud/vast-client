## Overview
This API client was generated by the [swagger-codegen](https://github.com/swagger-api/swagger-codegen) project.  By using the [swagger-spec](https://github.com/swagger-api/swagger-spec) from a remote server, you can easily generate an API client.

Generation command:
```bash
swagger-codegen generate -i ./vast_swagger_api.yaml --lang go
```

We also add an `AuthenticatingTransport` for conveniently authenticating requests, in `swagger/auth.go`.

Usage Example:
```go
transport := http.Transport{TLSClientConfig: &tls.Config{InsecureSkipVerify: true}}
authTransport := NewAuthenticatingTransport(&transport,
    "admin",
    "123456",
    "https://172.25.51.191/api/token/",
)

client := swagger.NewAPIClient(&swagger.Configuration{
    BasePath: "https://172.25.51.191/api/v2",
    HTTPClient: &http.Client{
        Transport: authTransport,
    },
})
```

- Build package: io.swagger.codegen.v3.generators.go.GoClientCodegen
